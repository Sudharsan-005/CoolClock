# Cool Clock

A stylish, interactive web-based digital clock with a lot of Custamization options.

## Features

-   **Customization**:
    -   **Fonts**: Cycle through 12 different retro and sci-fi fonts.
    -   **Colors**: Choose from preset neon colors or cycle through them.
    -   **RGB Mode**: Activate a dynamic RGB color cycling effect.
    -   **Background**: Upload your own custom background image.
    -   **Size & Brightness**: Adjust the clock size, opacity, and lightness.

-   **12/24 Hour Format**: Toggle between standard and military time.

## Controls

Control parameters using your keyboard:

| Key | Action |
| :--- | :--- |
| `0` | Cycle Font |
| `1` | Decrease Size |
| `2` | Increase Size |
| `3` | Cycle Color |
| `4` | Toggle RGB Cycle Effect |
| `5` | Decrease Brightness/Opacity |
| `6` | Increase Brightness/Opacity |
| `7` | Toggle 12/24 Hour Format |
| `8` | Toggle Digit Background Visibility |
| `9` | Toggle Date Display |
| `=` | Upload Background Image |
